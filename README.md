# Sales-Data-Analysis-MIS-Dashboard
This project analyzes retail sales data to identify revenue trends, product performance, regional growth, and inventory risks.  The objective was to simulate a real-world business environment by designing a relational database, performing SQL-based analysis, and building an MIS dashboard for decision-making  

# 📊 Sales & Inventory MIS Dashboard

## 📌 Project Overview

This project simulates a retail business scenario by analyzing sales and inventory data using SQL Server and Power BI. The dashboard enables users to monitor key business metrics, identify sales trends, evaluate regional performance, and manage inventory through an interactive one-page executive dashboard.

---

## 🎯 Project Objectives

- Analyze sales and profit performance
- Monitor inventory availability
- Track regional sales trends
- Identify top-performing products
- Support business decisions using interactive visualizations
