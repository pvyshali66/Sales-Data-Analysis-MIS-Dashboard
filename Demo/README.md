# 🎥 Dashboard Demo

## Project Walkthrough

This demo showcases the interactive capabilities of the **Sales & Inventory MIS Dashboard** developed using SQL Server, Power BI, DAX, and Excel.

### What you'll see in the demo

- Dashboard overview
- KPI Cards (Revenue, Profit, Margin, Low Stock Products)
- Interactive slicers (Date, Region, Category)
- Monthly sales trend analysis
- Regional sales comparison
- Product performance analysis
- Inventory monitoring
- Executive business insights
- Interactive filtering and cross-highlighting
